# Stephen新星防御 (Stephen Nova Defense)

一个经典的导弹指挥官风格塔防游戏，包含无尽模式、商店系统和金币收集。保护你的城市免受坠落火箭的攻击。

## 部署到 Vercel 指南

1. **上传到 GitHub**:
   - 在 GitHub 上创建一个新的仓库。
   - 将此代码推送到仓库：
     ```bash
     git init
     git add .
     git commit -m "Initial commit"
     git branch -M main
     git remote add origin <你的仓库URL>
     git push -u origin main
     ```

2. **连接到 Vercel**:
   - 登录 [Vercel](https://vercel.com)。
   - 点击 "Add New" -> "Project"。
   - 导入你的 GitHub 仓库。
   - Vercel 会自动识别 Vite 配置。
   - 点击 "Deploy"。

## 核心玩法
- **无限弹药**: 所有炮台现在拥有无限弹药。
- **实时显示**: 屏幕左上角实时显示得分，右下角实时显示金币和宝石。
- **金币收集**: 
  - 击毁敌方火箭有 5% 概率掉落金币。
  - **击杀奖励**: 每击败 10 个敌人触发一次金币掉落。
- **宝石系统**:
  - **蓝色BOSS**: 无尽模式下每增加 1000 分会生成一个巨大的蓝色导弹。
  - **击败奖励**: 击败蓝色BOSS必定掉落一颗宝石。
- **商店系统**: 
  - **金币区**: 购买永久强化（伤害 +2%，频率 +1%）。
  - **宝石区**: 购买临时道具。
    - **临时伤害**: 伤害提升 50%（持续 30 秒）。
    - **金币掉率**: 金币掉落概率提升 10%（持续 30 秒）。
  - **副作用**: 每次在商店购买任何物品，敌人的飞行速度将永久提升 **3.5%**（可无限叠加）。当前提升百分比实时显示在界面右侧。
- **视觉效果**:
  - **壮丽星空**: 全模式覆盖的动态星空背景。
  - **导弹外观**: 敌方投射物统一改为导弹外观。
  - **炸弹模型**: 我方拦截弹改为炸弹模型。
  - **锁子甲外观**: 炮台外壳采用金属锁子甲纹理。
- **无尽模式**: 
  - 达到 1000 分后可选择进入。
  - 目标积分为 20,000 分。
